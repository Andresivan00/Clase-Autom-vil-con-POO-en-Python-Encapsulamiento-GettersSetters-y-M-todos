# Clase Automóvil con POO en Python – Encapsulamiento, Getters/Setters y Métodos

Ejemplo educativo completo de **Programación Orientada a Objetos** que simula un automóvil con todos sus atributos y comportamientos reales.

### Qué hace este código
Crea una clase `Automovil` que representa un carro con las siguientes características:

- **Atributos** (marca, modelo, color, motor, velocidad actual/máxima, combustible, tipo, etc.)
- **Encapsulamiento** (atributos privados con `_`)
- **Getters y Setters** completos para leer y modificar cada atributo de forma controlada
- **Métodos de comportamiento reales**:
  - `acelerar()` → sube la velocidad
  - `desacelerar()` → baja la velocidad
  - `frenar()` → pone velocidad en 0
  - `tiempoLlegada(distancia)` → calcula cuántas horas tardaría en llegar
  - `imprimir()` → muestra toda la información del auto
